# flutter_js_runner

A Flutter JavaScript runner powered by `dart_qjs`.

Run it from this directory:

```bash
flutter run
```

Enter JavaScript code in the editor and press **Run** to evaluate it with
QuickJS. Results and JavaScript errors are shown in the output panel.
